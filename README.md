# Debug Assistance PenTest 🚀

> Esta herramienta fueron realizadas para probar el concepto de comunicacion con un pequeño asitente de Google, el Google Home mini. El codigo fue principalmente en python y fue realizado pidiendo apoyo a IA de google Gemini. Esto fue increíble.

> Lo que hacen sus dos primeras opciones son: Buscar en tu red asistentes conectados; Permite seleccionar (ESCANEAR) el asistente (TARGET) que vas a usar. 

>Y en tres opciones siguientes da la posibilidades enviar comandos por la red al asistente: Enviar un mensaje de la hora ofical (VOZ); Realizar un salto de volumenes (VOLUMEN); Traer informacion del dispositivo (RECON).

> Este script fue credo y probado en un sistema operativo Linux Ubuntu 26.04, con Visual StudioCode 1.115.0 y Python 3.13.9
